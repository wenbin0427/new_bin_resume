作品概述

  在大學時，我與朋友發現系上與學校缺乏一個能讓同學在線上認識彼此或進行課程交流的平台，因此我們共同創立了 「南應討論版_DIM社群📟」。這個平台的主要用途是為資管系的同學提供課程相關的專屬身分組，但該身分組需要經過審核，因此我們建立了這個驗證平台，讓同學能夠藉由此平台提交申請。

開發工具

  1. 前端：Vue、Bootstrap
  2. 後端：PHP、MySQL
  3. 伺服器：Google Cloud Platform VM個體 Ubuntu系統

負責項目

  網站架構規劃、網站頁面切版、後端 API 開發

主要功能

  提供同學線上申請、進度查詢、管理員可使用管理系統進行審核

備註

  目前系統上皆提供假資料進行瀏覽、管理員帳號密碼皆是：admin
